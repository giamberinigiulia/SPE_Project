# SPE_Project

## Installation

### Environment setup

- Windows:

```bash
$ python -m venv spe_env
$ spe_env\Scripts\activate 
```
- Linux:

```bash
$ python -m venv spe_env
$ source spe_env/bin/activate
```

### Installing dependency

```bash
$ pip install -r requirements.txt
```

### Launching application
```bash
$ python app.py
```
